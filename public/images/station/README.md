# Imagens da Estação

## LIDER

Coloque a foto do comandante da estação nesta pasta com o nome:
- `LIDER.jpg` ou
- `LIDER.png`

A imagem será exibida no modal "DADOS-ESTAÇÃO" no canto superior esquerdo.

### Recomendações:
- Tamanho recomendado: 200x200 pixels
- Formato: JPG ou PNG
- A imagem será exibida em formato quadrado
